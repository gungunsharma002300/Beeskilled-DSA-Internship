## ▶️ How to Compile & Run

Every `.cpp` file is a standalone program with its own `main()` function. To run any file:

```bash
g++ -std=c++11 -o output_name path/to/File.cpp
./output_name
```

Example:
```bash
g++ -std=c++11 -o bst_search "WEEK 3 – Trees & Hashing using C++/Assignments/01_BST_Search.cpp"
./bst_search
```

Each week's folder also has its own `README.md` with more specific details and sample inputs.
